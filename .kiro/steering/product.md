# Product Overview

This is a browser-based game project built during the AWS re:Invent workshop. The current implementation is "Flappy Kiro" - a Flappy Bird-style game featuring the Kiro logo as the player character.

## Purpose
- Educational workshop project for learning game development
- Collaborative game building experience with developers of all skill levels
- 2-hour workshop format focused on rapid prototyping and iteration

## Core Gameplay
- Side-scrolling obstacle avoidance game
- Player controls a character (Kiro logo) that must navigate through gaps in obstacles
- Score increases as obstacles are successfully passed
- Game over on collision with obstacles or screen boundaries

## Design Philosophy
- Start small, build smart - begin with simplest working version
- Iterative development with user feedback between tasks
- Creative audio and visual effects encouraged
- Retro pixel art aesthetic with modern polish
